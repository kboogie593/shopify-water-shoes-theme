## What To Check

| Check | Pass Signal | Fail Signal |
|---|---|---|
| 3-second hook | The first frame shows fur, a calm pet, a dust cup, or a before/after room result | First frame needs explanation |
| Visual proof | Viewers can understand the result without sound | Product value depends on specs |
| Repeatability | Many creators can film the same demo at home | Needs studio, actor, or complicated setup |
| Comment intent | Comments ask where to buy, whether it works for a breed, or how loud it is | Comments are mostly jokes or unrelated |
| Objection pattern | Repeated questions become FAQ/page topics | Comments show safety, noise, or trust concerns with no answer |
| Brand dependence | Generic product demo works | Only one famous brand or creator drives interest |

## First Manual Validation Rule

Before publishing aggressively, manually review at least 30 TikTok/Reels/Shorts posts across the search terms above. Record:

- Views
- Creator type
- Hook used
- Product shown
- Comment objections
- Whether the video is easy to remake
- Whether the product result is visible without narration
