## Page Creation Checklist In Shopify Admin

For each page above:

1. Create a Shopify page with the exact title and handle.
2. Choose the matching theme template in the page template dropdown.
3. Leave page body short or blank if the template already contains the main copy.
4. Add the page to the footer or a guide hub once all pages are created.
5. Submit the page URLs to Google Search Console after the theme is published.

## Claim Boundaries

Safe wording:

- Helps collect loose fur during routine brushing.
- Built for calmer at-home grooming routines.
- Useful for heavy-shedding homes that want less loose fur on sofas, rugs, bedding, and clothes.
- Should be introduced gradually for nervous pets.

Avoid until verified:

- Exact decibel claims.
- Exact suction power claims.
- Medical, skin-care, or anxiety-treatment claims.
- Claims that it stops shedding.
- Claims that every pet will tolerate it.
- Claims that it replaces professional grooming.
